# Smart Traffic Light Controller using Verilog HDL

## Overview
This project implements a Smart Traffic Light Controller using Verilog HDL. The system is designed using a Finite State Machine (FSM) approach to control traffic flow between North-South and East-West directions.

The project was simulated and verified using Xilinx ISE and ISim simulator.

---

## Features
- FSM-based traffic control
- North-South and East-West signal management
- Sequential RTL design
- Behavioral simulation verification
- Verilog HDL implementation

---

## Technologies Used
- Verilog HDL
- Xilinx ISE
- ISim Simulator

---

## FSM States
- NS_GREEN
- NS_YELLOW
- EW_GREEN
- EW_YELLOW

---

## Simulation
Behavioral simulation was performed using ISim simulator. Waveforms verified proper state transitions and traffic signal operation.
